## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript with serverless framework with docker.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Docker

### Docker Build

```bash
# Building Docker
$ npm run build:docker
```

### Docker Run

```bash
# Running Docker
$ npm run run:docker
```

### Docker Remove

```bash
# remove Docker
$ npm run remove:docker
```

## Lambda

### Deploy Serverless

```bash
# Deploy Serverless
$ npm run deploy:sls
```

### Remove Serverless

```bash
# Deploy Serverless
$ npm run remove:sls
```
